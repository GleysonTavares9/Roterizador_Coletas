===========================================
O QUE É FEATURE GRAPHIC?
===========================================

O Feature Graphic é uma imagem de BANNER que aparece:
- No topo da página do app na Google Play Store
- Em destaque quando o app é promovido
- Em categorias e coleções

TAMANHO OBRIGATÓRIO:
- Dimensões: 1024x500 pixels (exato!)
- Formato: PNG ou JPEG
- Tamanho máximo: 1 MB

===========================================
O QUE DEVE CONTER:
===========================================

✓ Logo do app (à esquerda ou centro)
✓ Nome do app: "Roterizacao Driver"
✓ Slogan/Descrição curta (opcional)
✓ Elementos visuais relacionados:
  - Ícone de caminhão
  - Mapa/GPS
  - Rotas

✗ NÃO PODE:
  - Texto muito pequeno (ilegível)
  - Imagens de baixa qualidade
  - Informações enganosas
  - Preços ou promoções

===========================================
OPÇÕES PARA CRIAR:
===========================================

OPÇÃO 1: CANVA (Recomendado - Fácil)
-------------------------------------
1. Acesse: https://www.canva.com
2. Crie conta gratuita
3. Pesquise: "Google Play Feature Graphic"
4. Escolha um template
5. Personalize:
   - Adicione o logo
   - Mude o texto para "Roterizacao Driver"
   - Ajuste cores (azul #2563EB)
6. Download: PNG, 1024x500

OPÇÃO 2: PHOTOSHOP/GIMP
-------------------------------------
1. Crie novo arquivo: 1024x500 pixels
2. Fundo: Gradiente azul (#2563EB → #1e40af)
3. Adicione o logo (ic_launcher-playstore.png)
4. Adicione texto: "Roterizacao Driver"
5. Adicione ícones de caminhão/mapa
6. Salve como PNG

OPÇÃO 3: POWERPOINT (Simples)
-------------------------------------
1. Abra PowerPoint
2. Configurar página: 1024x500 pixels
   (Arquivo > Configurar Página)
3. Fundo azul (#2563EB)
4. Insira o logo
5. Adicione texto "Roterizacao Driver"
6. Salvar como > PNG

OPÇÃO 4: FERRAMENTA ONLINE
-------------------------------------
https://www.photopea.com (Photoshop online grátis)
1. Novo projeto: 1024x500
2. Desenhe o banner
3. Download PNG
